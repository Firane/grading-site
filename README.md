## Déploiement GitHub Pages

1. Créer un repo GitHub (ex: `macbook-grading`)
2. Pousser ce dossier à la racine
3. Settings → Pages → Branch: main → /root → Save
4. Lien dispo : `https://[username].github.io/macbook-grading`
